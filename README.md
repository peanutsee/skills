# AI Agent Skills Library

A curated collection of mission-specific skills for AI coding agents. These skills enhance the capabilities of AI agents in specific domains, such as backend architectures, database optimization, or frontend frameworks.

## 🚀 Available Skills

- **[supabase-expert](file:///Users/peanutsee/Desktop/projects/skills/supabase-expert/SKILL.md)**: Architect and optimize Supabase integrations for full-stack applications.

## 🛠️ How to Use

To use these skills with your AI agent:
1. Copy the contents of the desired `SKILL.md` file.
2. Provide it to your AI agent as system instructions or as a reference for a specific task.
3. If the agent supports a dedicated skill system, point it to the directory containing the `SKILL.md`.
